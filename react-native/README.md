## Install

`npm i @reaction-eslint/eslint-config-react-native -D`

or

`yarn add @reaction-eslint/eslint-config-react-native --dev`



### Usage

```json
{
  "env": {
    "jest": true
  },
  "extends": [
    "@reaction-eslint/react-native"
  ]
}
```
