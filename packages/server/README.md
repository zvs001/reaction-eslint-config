## Description

Set of rules for typescript + mocha tests project


## Install

`npm i @reaction-eslint/eslint-config-server -D`

or

`yarn add @reaction-eslint/eslint-config-server --dev`



### Usage

```json
{
  "env": {
    "es2017": true,
    "node": true,
    "mocha": true
  },
  "extends": [
    "@reaction-eslint/server"
  ]
}
```
